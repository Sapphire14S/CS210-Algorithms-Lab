## Laboratory Overview

This repository contains implementations and practical assignments related to searching algorithms, divide and conquer, dynamic programming, greedy methods, graph algorithms, shortest path techniques, minimum spanning trees, and optimisation problems.

---

## Labs Overview

| Lab | Concepts Covered |
|-----|------------------|
| Lab&nbsp;01 | Binary search applications, cyclically sorted arrays, interpolation search, fixed-point search problems |
| Lab&nbsp;02 | Median of two sorted arrays, searching in sorted matrices, mountain array search problems |
| Lab&nbsp;03 | Dynamic programming, Longest Increasing Subsequence (LIS), Coin Change problem, Frog Jump minimum energy problem |
| Lab&nbsp;04 | Divide and conquer algorithms, missing number detection, maximum product subarray, majority element detection, Median of Medians, Karatsuba multiplication, interval overlapping |
| Lab&nbsp;06 | Greedy algorithms, optimal file merging, interval covering problems, interval partitioning and scheduling |
| Lab&nbsp;07 | Dynamic programming on paths and strings, minimum-cost path counting, lexicographically smallest paths, banana collection problems, maximum non-consecutive profit |
| Lab&nbsp;08 | Longest Common Subsequence (LCS), Longest Palindromic Subsequence (LPS), Maximum Sum Increasing Subsequence (MSIS), Longest Common Increasing Subsequence (LCIS), dynamic programming optimisation |
| Lab&nbsp;09 | Graph representations, adjacency matrix/list conversion, graph type detection, graph transpose, path counting using matrix multiplication |
| Lab&nbsp;10&nbsp;–&nbsp;11 | BFS and DFS traversals, connected components, bipartite graph checking, cycle detection, topological sorting, strongly connected components, articulation points, bridges, grid-based BFS problems |
| Lab&nbsp;12 | Minimum Spanning Trees (MST), Kruskal’s algorithm, Prim’s algorithm, cut property, cycle property, uniqueness and sensitivity of MSTs |
